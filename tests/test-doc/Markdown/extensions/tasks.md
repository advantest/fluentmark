### [Extension: `task_lists`](https://pandoc.org/MANUAL.html#extension-task_lists)

Thanks to pandoc we can create task lists with checked or unchecked tasks.

---

Examples:


Some task list:

- [ ] an unchecked task list item
- [x] checked item


Another task list:

* [ ] an unchecked task list item
   * [x] checked item
   * [ ] unchecked
* [ ] another unchecked item
